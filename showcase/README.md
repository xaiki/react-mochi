# Mochi UI showcase

Minimal static showcase of all `@mochi/ui` components. No Vite or dev server required.

## Build

From the **mochi package** root (one level up):

```bash
npm run build          # build the library first
cd showcase && npm install && npm run build
```

## View

Open `showcase/index.html` in a browser (file:// or any static server). The left column lists component groups; the right side shows live demos.
